# Students Social Media Addiction Analysis Using Power BI

## Project Overview
This project analyzes social media usage patterns among students and studies the impact of excessive usage on academic performance, sleep patterns, and mental health.

The analysis is performed using Microsoft Power BI by creating interactive dashboards and visualizations.

## Tools & Technologies
- Microsoft Power BI
- Power Query (Data Cleaning)
- DAX (Data Analysis Expressions)
- Excel / CSV Dataset

## Dataset Information
The dataset contains information about:
- Daily social media usage hours
- Addiction score
- Academic performance
- Sleep duration
- Mental health indicators

## Dashboard Insights
The Power BI dashboard provides the following insights:

- Total number of students analyzed
- Average daily social media usage
- Distribution of addiction levels
- Relationship between social media usage and academic performance
- Impact of sleep hours on mental health

## Project Files
- Dataset
- Power BI Dashboard (.pbix)
- Project Presentation (PDF)
- Dashboard Screenshots

## Conclusion
The analysis shows that excessive social media usage may negatively impact students' academic performance and overall well-being. The dashboard helps in identifying behavioral patterns through interactive visualizations.
